# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a server-persisted, per-user dark mode with a header toggle button using Tailwind's `dark:` variant.

**Architecture:** A new DB column `dark_mode` on `users` is read at login and returned in every user response. `PUT /api/users/me/preferences` persists changes. The frontend reads the preference on boot, applies `dark` to `<html>`, and syncs on toggle. All color swaps are Tailwind `dark:` class pairs in `index.html`.

**Tech Stack:** Go (chi router), SQLite, Alpine.js, Tailwind v4 standalone CLI

- [ ] **Step 1: Add HandleUpdatePreferences to user service**

`internal/user/user.go` — add method (before the closing brace of the file):
```go
func (s *Service) HandleUpdatePreferences(w http.ResponseWriter, r *http.Request) {
    u := auth.UserFromContext(r.Context())
    var req struct {
        DarkMode bool `json:"dark_mode"`
    }
    if err := json.NewDecoder(r.Body).Decode(&req); err != nil {
        http.Error(w, "bad request", http.StatusBadRequest)
        return
    }
    darkModeInt := 0
    if req.DarkMode {
        darkModeInt = 1
    }
    if _, err := s.db.Exec(`UPDATE users SET dark_mode = ? WHERE id = ?`, darkModeInt, u.ID); err != nil {
        http.Error(w, "internal error", http.StatusInternalServerError)
        return
    }
    w.WriteHeader(http.StatusNoContent)
}
```

`internal/api/router.go` — add before `r.Put("/api/users/me/password", ...)`:

```js
async init() {

        const r = await fetch('/api/auth/me');
        if (r.ok) {
```

```js
    this.events = [];
    this.darkMode = false;
    document.documentElement.classList.remove('dark');

```

Add `toggleDark` method after `logout()`:
```js
async toggleDark() {
    this.darkMode = !this.darkMode;
    document.documentElement.classList.toggle('dark', this.darkMode);
    await fetch('/api/users/me/preferences', {
        method: 'PUT',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ dark_mode: this.darkMode }),
    });
},
```
